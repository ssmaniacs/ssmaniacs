オブジェクトファインダー(2017/10/16更新：ハロウィンアップデート)
各シーンのオブジェクトの隠れ場所を確認できます。画像が大量なのでかなり重いです。
Chrome, Firefox, Safari, Internet Explorerあたりの新しいバージョンなら動くはず。
小画面スマホ、モバイル回線などでの使用にはオブジェクトファインダー低解像度版もどうぞ。
画像が若干不鮮明になりますが、環境によってはロード時間が3分の1～4分の1程度に短縮されます。

以下の機能は ssmaniacs.github.io のサイトでは利用できません。ブラウザからSSサーバにアクセスするためのスクリプトが使用できないためです。 これらを使用する場合は github 以外のサイトを使用してください。

ミラーサイト

フリーホスティングサイトを利用しているため、運用が不安定な場合があります。その時その時で生きているところを使ってください。
  • http://ssmaniacs.php.xdomain.jp/
    比較的安定していますが、反応が遅い場合があります。スマホ等、ブラウザ環境によっては広告が表示されます。
  • http://ssmaniacs.000webhostapp.com/
    反応は比較的速いですが、時々サーバが止まります。
  • https://ssmaniacs.github.io/ssmaniacs/
    非常に安定しており速度も充分ですが、シークレットサンタなどSSサーバへのアクセスが必要な機能は動作しません。
このサイトに載っている情報・コードなど、全て自由に使ってください。
リンク、転載、改変、再配布など、お好きにどうぞ。
アイコンなどのゲーム画像の利用方法についてはゲーム自体の許諾条項に準じるものとします
(画像はすべてゲーム本体から抽出したものなので）。
