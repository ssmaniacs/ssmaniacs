オブジェクトファインダー (2017/6/9更新)
各シーンのオブジェクトの隠れ場所を確認できます。画像が大量なのでかなり重いです。
Chrome, Firefox, Safari, Internet Explorerあたりの新しいバージョンなら動くはず。
小画面スマホ、モバイル回線などでの使用にはオブジェクトファインダー低解像度版もどうぞ。
画像が若干不鮮明になりますが、環境によってはロード時間が3分の1～4分の1程度に短縮されます。
このサイトに載っている情報・コードなど、全て自由に使ってください。
リンク、転載、改変、再配布など、お好きにどうぞ。
