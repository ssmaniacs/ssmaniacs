オブジェクトファインダー(2017/12/22更新:クリスマスアップデート)
各シーンのオブジェクトの隠れ場所を確認できます。画像が大量なのでかなり重いです。
Chrome, Firefox, Safari, Internet Explorerあたりの新しいバージョンなら動くはず。
小画面スマホ、モバイル回線などでの使用にはオブジェクトファインダー低解像度版もどうぞ。
画像が若干不鮮明になりますが、環境によってはロード時間が3分の1～4分の1程度に短縮されます。

以下の機能ではブラウザからSSサーバにアクセスするためぶ外部スクリプトまたは外部プロキシを使用しています。
ホスティングサーバまたは外部プロキシの状態によっては利用できないことがあります。

シークレットサンタ
自分のアカウントへウィッシュリスト内のアイテムを送ることができます。
クローンプロファイル
ゲーム進行データを異なるデバイス間でコピーすることができます。
プロファイルビューア
ギフトボックス、友達リスト、所持品リスト、各シーンの進行状況を確認することができます。
2017年10月のアップデート（ハロウィンイベント）から、端末とSSサーバ間の通信方法が変更されました
2017年10月16日時点では従来の通信方法も利用できていますが、近い将来（アップデートが行き渡ったぐらい？） 従来の通信方法が使用できなくなることが予想されます。
従来方式の通信ができなくなった時点でシークレットサンタ、クローンプロファイルおよびプロファイルビューアは 利用できなくなる可能性がありますので、 あらかじめご了承ください。
